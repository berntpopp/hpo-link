# hpo-link Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `hpo-link`, a deterministic MCP server serving HPO ontology + HPOA gene/disease annotations from a locally-built SQLite DB that is published to GitHub as a versioned Release artifact.

**Architecture:** Fork the proven `mondo-link` `-link` server (layered: ingest ‚Üí data DAO ‚Üí services ‚Üí mcp), rename `mondo`‚Üí`hpo`, swap the data layer for HPO (hp.json obographs + HPOA TSVs, new schema, new parsers), and add the net-new annotation tools, prebuilt-artifact distribution (`release.py`), and `build-data.yml` CI.

**Tech Stack:** Python 3.12, uv, hatchling, FastMCP (`mcp[cli]>=1.27`, `fastmcp>=3.2`), FastAPI + uvicorn, pydantic v2 + pydantic-settings, httpx, structlog, orjson, typer; SQLite + FTS5; ruff + mypy + pytest.

## Global Constraints

- Python 3.12 minimum; package dir `hpo_link/` (underscore); `uv` only (never pip).
- Build backend hatchling. Entry points: `hpo-link = "server:main"`, `hpo-link-mcp = "mcp_server:main"`, `hpo-link-data = "hpo_link.ingest.cli:main"`.
- FastMCP facade `FastMCP(name="hpo-link", instructions=..., mask_error_details=True)`.
- All tools `hpo_`-prefixed (except `get_server_capabilities`, `get_diagnostics`); `annotations=READ_ONLY_OPEN_WORLD`; `response_mode` default `"compact"`; projection lives in the service layer.
- Every response carries `_meta` (`{tool, request_id, next_commands}`), `recommended_citation`, and `hpo_version`.
- Errors returned as dicts, never raised across the MCP boundary. Error codes: `not_found`, `ambiguous_query`, `invalid_input`, `data_unavailable`, `rate_limited`, `upstream_unavailable`, `internal_error`.
- ruff line-length 100; mypy strict; ‚â§500 lines/file (`scripts/check_file_size.py`); coverage gate ‚â•70%.
- Logging via structlog ‚Üí stderr only; no stdout prints in library code (stdio MCP purity).
- License/citation: display HPO version on output; cite K√∂hler S, et al. NAR 2024;52(D1):D1333‚ÄìD1346; custom HPO license `https://hpo.jax.org/app/license`. Research-use-only.
- Data sources (version-pinned, default edition `hp.json`):
  - latest-tag discovery: `https://api.github.com/repos/obophenotype/human-phenotype-ontology/releases/latest`
  - ontology: `http://purl.obolibrary.org/obo/hp/releases/{DATE}/hp.json`
  - HPOA: `‚Ä¶/releases/{DATE}/{phenotype.hpoa,genes_to_phenotype.txt,phenotype_to_genes.txt,genes_to_disease.txt}`
- `SCHEMA_VERSION = 1`.

```python
# hpo_link/identifiers.py
"""HPO / gene / disease identifier normalization."""
from __future__ import annotations
import re

_HP_RE = re.compile(r"(?:HP[:_])?0*(\d{1,7})$", re.IGNORECASE)
_IRI_RE = re.compile(r"HP[_:](\d{7})$", re.IGNORECASE)

def normalize_hpo_id(raw: str | None) -> str | None:
    if not raw:
        return None
    s = raw.strip()
    m = _IRI_RE.search(s) or _HP_RE.match(s)
    if not m:
        return None
    return f"HP:{int(m.group(1)):07d}"

def iri_to_curie(iri: str) -> str:
    return normalize_hpo_id(iri) or iri

def is_hpo_id(s: str | None) -> bool:
    return bool(s and re.match(r"^HP:\d{7}$", s.strip(), re.IGNORECASE))

def normalize_gene(raw: str) -> tuple[str, str]:
    s = (raw or "").strip()
    low = s.lower()
    if low.startswith("ncbigene:"):
        return ("ncbi", s.split(":", 1)[1])
    if s.isdigit():
        return ("ncbi", s)
    return ("symbol", s.upper())

def normalize_disease_id(raw: str) -> str:
    s = (raw or "").strip()
    if ":" in s:
        prefix, rest = s.split(":", 1)
        return f"{prefix.upper()}:{rest}"
    return s
```
